# Designo Aisys

Designo Aisys is a white-label product design workspace for turning an idea into an implementation-ready brief.

The app helps teams define:

- product vision and constraints
- data entities and rules
- visual system direction
- product sections, screens, and states
- a handoff package in JSON or Markdown

## Run locally

This first version is a static web app and does not require a package manager.

Open `index.html` in a browser, or serve the folder with any static server.

## Project structure

```txt
index.html
src/
  app.js
  styles.css
```

## Notes

This codebase is an original white-label implementation. It does not copy source files, interface text, branding, or assets from any third-party product.
